MagicFlow in RapinnoAssist

MagicFlow is a powerful AI tool within RapinnoAssist that enables workflow creation simply by providing a title and description. With MagicFlow, you can build workflows in seconds.

Select Workflows​

Navigate to Workflows, which will direct you to the Workflows screen.

This platform allows you to create workflows automatically by providing a prompt.

Create a New Workflow​

  1. Click on the + New Workflow icon to initiate a new workflow for your AI bot.

  2. The Add New Workflow pop-up screen will appear, allowing you to configure your workflow settings.

  3. In the Add New Workflow screen, select Magic Flow.

  4. Provide the necessary details in the following fields to create a Magic Flow instantly:

    OptionDescription
    Add a TitleEnter the workflow title in the text box where "Title" is the placeholder.
    Describe itProvide a detailed description of the workflow's purpose. The placeholder will read "Describe it".

Chatbot Actions​

Welcome Messages​

The welcome message is the initial interaction of the AI bot, automatically starting the conversation with the user.
Example: "Welcome back! How may I help you today?"

Questions​

After greeting, the AI bot will ask questions to understand the user’s intent.
Example: "Please let me know if you have any questions about our product."

Outputs​

After posing a question, the chatbot will display multiple-choice options for the user. Each option leads to a different response path and information-gathering process.
Example: "Select a topic from the options - Send Money, Transaction Status, Insurance."

Responses​

When the user selects an option, the AI bot will reply with a confirmation message. You can create customized responses for each scenario.
Example: "I can help with that. Please provide your transaction ID."

End Messages​

Once the AI bot collects the necessary information, it will conclude the interaction with an end message.
Example: "Click OK to view the transaction status."

Realign Nodes​

If your workflow appears cluttered or difficult to follow after creation, you can instantly improve its visual clarity by using the Realign Nodes feature. With just a click, this tool automatically organizes the nodes, making the workflow structure cleaner and more visually intuitive. This helps you understand and manage the flow with ease, ensuring each step is easily identifiable and accessible.

After creating the workflow, you can click on Realign Nodes, located next to the Back button, to automatically tidy up the layout. This feature optimizes the arrangement of your workflow, making it more organized and easier to understand at a glance.

Magic write​

Within each Interaction node, you'll find a feature called Magic Write. When you click on a node, magic write appears, allowing you to enter the text that will be displayed in the chatbot. This AI-powered tool generates suggested text, helping you craft responses efficiently and effectively. With Magic Write, you can quickly create engaging, AI-driven dialogue for your chatbot, ensuring each interaction is meaningful and relevant.

Using Magic Write for Text Responses​

This guide will walk you through using Magic Write to generate and select text responses in the interaction node.

Steps:​

  1. Locate Magic Write:

    • In the interaction node, navigate to the Text response area.
    • Above this text area, you will see the Magic Write button.

  2. Click on Magic Write:

    • Click the Magic Write button to open text suggestions for your response.

  3. Review Suggested Responses:

    • After clicking, Magic Write will suggest several text responses.
    • Scroll through the options and choose the one that best matches your intent.

  4. Select and Enhance:

    • Once you've selected a suggestion, you can customize it further to align perfectly with your needs.
