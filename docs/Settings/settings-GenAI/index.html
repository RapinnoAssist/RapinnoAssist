GenAI

To leverage the advanced capabilities of Large Language Models (LLMs) and Generative AI features in your application, you need to configure the integration using Gen AI.

There are 2 options to intergrate.

  1. Enterprise
  2. Open Source

Enterprise Models​

In Enterprise, we can integrate with premium platforms like,

  1. Azure OpenAI
  2. OpenAI
  3. Gemini
  4. Anthropic

Azure OpenAI​

Firstly,In Azure OpenAI for authorization purpose you have enter the details for two fields. One is API key and another one is Sub domain name.

Next, we have to choose the models depending on the purpose .There are two models.

  1. Language Models
  2. Embedding Models

Language Models​

The language model processes the fetched documents to generate a coherent and contextually relevant answer to the user's question.

Embedding Models​

Embedding models create embeddings of data, stored in a vector database. When a user asks a question, it is embedded, and similar documents are fetched from the vector database. These documents are processed by a Generative LLM to generate the answer.

You have to full the deployment ID field in which ever model you choose.
