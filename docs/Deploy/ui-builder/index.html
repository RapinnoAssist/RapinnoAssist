UI Builder

You can configure the chat widget settings such as the bot appearance and bot avatar in this module. You can have a real-time preview of the configured chat widget. Click UI Builder , it directs you to the Chat Widget Builder screen.

Choose Instance​

Select an instance from the list to configure the respective chat widget. Once you select an instance, it displays the screen like this

Create the instance by clicking the dropdown then select default to create an default instance

After selecting the instance click Next.

Design​

Basic​

Once you select an instance, you will be directed to the Design screen automatically as shown above. Provide the below basic details for to design the chat widget.

OptionDescription
Bot NameSpecify the name for the AI bot.
Bot MessageWrite a welcome message for users. For instance, "Hi! I'm here to help you with any questions you have."
Bot AvatarProvide the image for the bot avatar, either as Base64 code or as a URL link.
Launcher IconUpload the launcher icon image using Base64 code or a URL link.
Header LogoInclude the header logo image using Base64 code or a URL. The default is the Cogniwide logo.
Choose Header TypeSelect the type of header. Options are Banner for a single image or Carousel for multiple images.
Banner URLProvide the image for the banner header, either as Base64 code or as a URL. This option is visible if you choose Banner.
Banner TextSpecify the action name to trigger when the banner image is clicked. Example, /getquotation. This appears if Banner is selected.
URLProvide images for the carousel header, either as Base64 code or as a URL. This is visible if Carousel is chosen.
TextEnter the action name to trigger when clicking on a carousel image. Example, /getquotation. This field appears if Carousel is selected.

Theme​

You can define the position and color of the bot in this section. Click the Theme tab, it displays the below screen.

OptionDescription
Bot AlignmentChoose the left alignment option to position the chat widget on the left side of the screen. Alternatively, select the right alignment option to place it on the right side.
Bot ColorsChoose the primary and secondary colors to customize the bot's appearance.

Advanced settings can be done in this section. Click the Advanced tab, it directs you to the below screen.

Communication Method

There can be two ways of Communication.

  1. REST
  2. SOCKET

REST mode contains only the BOT URL

SOCKET contains the Socket URL and the Socket Path you can give the URL and the Path in the text box
