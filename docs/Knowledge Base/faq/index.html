FAQ

The FAQ section provides instant answers to common questions, reducing the need for customers to contact support. This functionality allows you to associate specific questions and user intents with each product, refer to Intents.

Select FAQs under the Knowledge Base module to access the FAQ screen as shown below.

Add FAQ​

Add New or Select Pre-existing FAQ

Enter a new FAQ title or select a preexisting intention from the list, then click the Add icon .

OptionDescription
TitleEnter the title of the FAQ
FAQUse pre-existing FAQs
Contextual DisplayEnter an optional FAQ title relevant to your product, e.g., more options on bill payments.
LanguageSelect the primary and secondary languages from the list.
DescriptionProvide a brief description related to the FAQ, e.g., common questions on paying various bills.
Add and DeleteClick the Save button to save the FAQ. To delete an existing FAQ, click the Delete button .

Upload CSV File​

Click the Upload CSV button and upload the required file in the supported format.

Update or Delete FAQ​

To update, click the existing product title, modify the details, and click the Save button . To delete, click the Delete icon for the respective FAQ and proceed with deletion.
